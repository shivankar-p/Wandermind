## Steps to run

Clone the repository
```
git clone https://github.com/shivankar-p/Wandermind.git
cd /path/to/repo/
```

Run Frontend
```
npm start
```

Run Backend
```
cd backend
python3 main.py
cd server/src
npm i
node index.ts
```
